Difference b/w 'var', 'let', 'const'

var

The scope is global when a 'var' variable is declared outside a function.
'var' is function scoped when it is declared within a function.
'var' can be updated and re-declared

let

'let' is block scoped
A block is a chunk of code bounded by {}.
A block lives in curly braces
A variable declared in a block with 'let' is only available for use within that block
'let' can be updated but not re-declared

const

Variables declared with the 'const' maintain constant values
'const' declarations can only be accessed within the block they were declared
'const' cannot be updated or re-declared
